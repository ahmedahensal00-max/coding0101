# كيفية الحصول على Gemini API Key مجاناً

## الخطوات:

### 1. اذهب إلى Google AI Studio
افتح الرابط: [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)

### 2. سجل الدخول
- استخدم حساب Google الخاص بك
- إذا لم يكن لديك حساب، أنشئ واحداً مجاناً

### 3. احصل على API Key
- اضغط على **"Create API Key"** أو **"Get API Key"**
- اختر مشروع Google Cloud (أو أنشئ مشروعاً جديداً)
- انسخ المفتاح (يبدأ بـ `AIza...`)

### 4. أضف المفتاح إلى المشروع
افتح ملف `.env.local` واستبدل:
```
GEMINI_API_KEY=YOUR_GEMINI_API_KEY_HERE
```

بـ:
```
GEMINI_API_KEY=AIzaSy...  (المفتاح الخاص بك)
```

### 5. احفظ الملف واختبر
- احفظ ملف `.env.local`
- شغل السيرفر: `npm run dev`
- جرب الشات بوت!

## الحصة المجانية:
- ✅ **1500 طلب/يوم** مجاناً
- ✅ **1 مليون token/دقيقة**
- ✅ بدون بطاقة ائتمان

## ملاحظات مهمة:
- ⚠️ لا تشارك المفتاح مع أحد
- ⚠️ ملف `.env.local` محمي ولن يُرفع إلى Git
- ✅ المفتاح مجاني تماماً ولا يحتاج دفع
